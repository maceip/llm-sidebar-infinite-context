Native overlay companion: current status

The extension can connect to a Rust native companion service with reconnect and heartbeat support after you build the native binaries and run the installer. The sidebar status pill will then show whether the bridge is connected and whether overlay rendering is available on your platform.

  • Visible HUD / overlay target: macOS and Windows.
  • Linux currently supports the native bridge and diagnostics, not a visible overlay window.
  • The bundled Puppeteer harness installs the native companion in an isolated profile and proves browser memory plus heartbeat behavior in headless Chrome.

Privacy first. Your API key and conversations stay in your browser. No data is sent to any server other than the Gemini API.
